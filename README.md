# 🔐 AuthProject - Full Django Authentication System

Modern Django authentication system with JWT, OTP, Roles, and beautiful Tailwind UI.

## ✨ Features

- User Registration
- Login / Logout
- OTP Login
- Profile Management
- Admin Dashboard
- JWT Authentication
- Dark Mode
- Responsive UI
